# PHP Hangman Game

A server-side Hangman game built using PHP. This project demonstrates core programming logic, session handling, and user interaction through a web-based interface.

---

## Tech Stack
- PHP
- HTML, CSS

---

## Features
- Server-side game state management
- Word guessing logic and input validation
- Difficulty selection and score tracking
- Dynamic feedback for correct and incorrect guesses

---

## Notes
This project focuses on backend logic and problem-solving using PHP.
